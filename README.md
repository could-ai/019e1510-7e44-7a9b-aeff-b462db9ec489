# M.Com Dissertation Viewer

This Flutter project provides an interactive digital viewer for the M.Com 4th Semester dissertation. It was created to provide a structured, easy-to-read, and accessible interface for the research content before it is compiled into a final printable word document.

## Overview
**Title:** The "Gamification" of Trading: An Empirical Analysis of Mobile App Interface Design on the Risk-Taking Behavior of Gen Z Investors  
**Author:** G GAUTAM Rao  
**Course:** M.Com 4th Semester  

## Features
- **Responsive Layout:** The application adapts automatically to mobile, tablet, desktop, and web views, ensuring a smooth reading experience on any device.
- **Chapter Indexing:** A clear table of contents makes it easy to navigate through Chapters 1 to 5.
- **Copy Functionality:** Inside each chapter, a simple copy button allows the reader or author to copy the exact text to the clipboard so it can easily be pasted into Microsoft Word for formatting and printing.
- **Structured Content:** The underlying data separates chapters into a cleanly maintained list of maps, making it easy to add mock analysis, edit text, or format headers.

## Getting Started
To view or run the application:
1. Ensure you have the Flutter SDK installed.
2. Clone the repository and navigate to the root directory.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application.

If you are using the web viewer, simply reload to see the latest text modifications in `/lib/data/dissertation_data.dart`.
